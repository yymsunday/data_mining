数据挖掘分析实训平台 - 模块化架构实现
一个基于 Streamlit 的模块化数据挖掘分析实训平台，采用"主程序调用 + 模块核心逻辑"的架构设计，实现代码复用和逻辑分离。

🎯 项目理念
核心架构原则：

主程序 (data.py)：负责"调用 + 优化 + 展示"

功能模块 (stepxx.py)：负责"核心逻辑 + 返回原始结果"

数据流：模块返回原始数据 → 主程序优化处理 → 可视化展示

🚀 快速开始

🔧 模块化架构说明
主程序 (data.py) 职责：
Web界面构建：侧边栏导航、页面布局

模块调用：导入并执行对应 stepxx 模块

结果优化：将原始数据转换为用户友好格式

可视化展示：图表、表格、统计信息展示

代码展示：实时显示核心逻辑代码

功能模块 (stepxx.py) 职责：
核心算法：实现具体的金融计算逻辑

数据处理：读取、清洗、计算原始数据

结果返回：返回标准化的数据结构
